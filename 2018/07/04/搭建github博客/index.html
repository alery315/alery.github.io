新建repository

新建github repository,名称为<blog_name>.github.io,<blog_name>即是自己博客的名字,项目名称为访问的域名.(创建完成后可能需要等一会)

在项目的settings页面可以看到GitHub Pages一项,可以看到是否正常工作.

我的hexo theme

我使用的是hexo-theme-melody,使用也很方便,github页面都有讲到,还有一份完整的文档.

只需要将该项目clone到themes文件夹,然后改下主文件夹下的_config.yml,搜索theme,修改默认的theme为melody即可,执行hexo s -g就可以看到效果.
