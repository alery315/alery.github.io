Supervised Learning

在监督学习中,我们有一个数据集,我们知道正确的输出是什么样的,并且认为输入与输出之间存在着某种关系.

监督学习分为回归问题和分类问题.

回归问题就是在连续的输出中预测结果,我们需要尝试将输入变量映射到某个连续函数.

分类问题就是在离散输出中预测结果,我们需要将输入变量映射到离散类别.

举个例子:

  • 回归问题,鉴于一个人的照片,根据照片来预测他们的年龄.
  • 分类问题,鉴于患有肿瘤的患者,预测出肿瘤是恶性还是良性的.

Unspuervised Learning

无监督学习可以让我们在对结果是什么样了解的很少或者根本不知道的时候解决问题,我们可以从数据中导出结构,我们不一定知道变量对于结果的影响.

我们可以通过基于数据变量之间的关系聚类来推导出这个结构.

在无监督学习中,没有基于预测结果的反馈.

举个例子:

  • 聚类:收集1000000个不同基因的集合,并找到一种方法将这些基因自动分组成某种程度上相似的或者通过不同的变量连接在一起的.比如寿命,出生地,角色等等.
  • 非聚类(分离):”鸡尾酒会算法”运行你在混乱的环境中查找结构.(即在鸡尾酒会上识别个人声音和音乐)
