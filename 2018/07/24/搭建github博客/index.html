新建repository

新建github repository,名称为<blog_name>.github.io,<blog_name>即是自己博客的名字,项目名称为访问的域名.(创建完成后可能需要等一会)

在项目的settings页面可以看到GitHub Pages一项,可以看到是否正常工作.
