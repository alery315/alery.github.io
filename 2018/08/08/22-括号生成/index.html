括号生成

题目:

给出 n 代表生成括号的对数,请你写出一个函数,使其能够生成所有可能的并且有效的括号组合.

例如，给出 n = 3，生成结果为:

暴力思路:

生成所有的序列,然后判断有效性,一共有2^(2*n)个.

采用递归生成序列,长度为n的序列就是’(‘加上所有长度为n-1的序列,以及’)’加上所有长度为n-1的序列.

感觉递归这个东西就很神奇.

回溯方法

在知道序列有效时候才去添加’(‘和’)’,而不是像暴力方法那样每个都添加然后最后判断,可以根据当前左右括号的数目来判断有效性.

当左括号小于N时候,可以添加左括号;当右括号数目小于左括号时候,才能添加右括号.

我觉得这个方法容易理解,而且时间复杂度不高.
