多元线性回归

不再是一个变量影响结果,而是多个,一般的比如n,

hθ(x) = θ0 + θ1X1 + θ2X2 + … + θnXn

写为向量表示为(INV表示转置)

X = INV[x0,x1,…xn]

θ = INV[00,θ1,…θn]

hθ(x) = INV(θ) * X

使用梯度下降解决多元线性回归

其实和一元线性回归本质是一样的,偏导数基本类似,就是多个n个参数而已.(x0恒等于1)

特征缩放

如果可以保证变量都在相似的范围内,就可以让梯度下降法能够更快地收敛.

尽量将变量xi的值放缩到-1 <= xi <= 1 这个范围内,乘除倍数,差不多也可以.

如上图所示,特征缩放后可以很快地收敛,而原来的大范围则会一直抖动.

学习速率α

  • 调试梯度下降:在x轴上绘制具有迭代次数的图.现在绘制成本函数,J(θ)超过梯度下降的迭代次数.如果J(θ)增加,那么你可能需要减少α。

  • 自动收敛测试:如果在一次迭代中,J(θ)减少的值小于E(一个很小的值,比如10^-3),那么我们认为已经收敛了.但是在实际应用中我们很难选取这个阈值.

总结:

  • 如果α太小了,收敛会很慢.

  • 如果α太大了,每次迭代可能不会减小,因此可能不会收敛.

多项式回归

我们可以把曲线改为二次,三次或者平方根的形式,比如hθ(x) = θ0 + θ1x1,我们可以根据x1添加一个变量x2 = x1^2,变成hθ(x) = θ0 + θ1x1 + θ2x1^2,或者三次的形式hθ(x) = θ0 + θ1x1 + θ2x1^2 + θ3x1^3,或者平方根的形式hθ(x) = θ0 + θ1x1 + θ2squa(x1).
