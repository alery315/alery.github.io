多元线性回归

不再是一个变量影响结果,而是多个,一般的比如n,

hθ(x) = θ0 + θ1X1 + θ2X2 + … + θnXn

写为向量表示为(INV表示转置)

X = INV[x0,x1,…xn]

θ = INV[00,θ1,…θn]

hθ(x) = INV(θ) * X

使用梯度下降解决多元线性回归

其实和一元线性回归本质是一样的,偏导数基本类似,就是多个n个参数而已.(x0恒等于1)

特征缩放

如果可以保证变量都在相似的范围内,就可以让梯度下降法能够更快地收敛.

尽量将变量xi的值放缩到-1 <= xi <= 1 这个范围内,乘除倍数,差不多也可以.
